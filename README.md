# Paint

A simple paint app using TKinter

## Features
  * pen tool
  * brush tool
  * eraser tool
  * export tool
  * brush size tool
  * color picker tool


## Libraries used
  * TKinter
  * Pillow

## Installation
  ```
  git clone gh repo clone Vishnuxx/TKinterHack
  cd Paint
  python3 main.py
  ```
